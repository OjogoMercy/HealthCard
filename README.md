**HealthCard** is a mobile health (mHealth) application designed to bridge the gap in postnatal care and child nutrition monitoring in Nigeria. 

###  Key Features
* **Digital NPI Tracker:** A digital replacement for the physical immunization "Green Card" with automated due-date logic and local reminders.
* **Nutritional Monitoring:** A simplified UX using sliders and food-group icons, making it easy for mothers to log meals regardless of literacy level.
* **Growth Tracking:** Tools for monitoring developmental milestones and postnatal follow-up.
* **Persona-Driven Design:** Optimized for both high-end smartphones (Career Moms) and low-bandwidth/offline environments (Rural/Displaced Moms).
